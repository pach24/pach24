### Enterprise-Grade Android Architecture

This invoicing application bridges the gap between academic theory and **industry standards**, showcasing a scalable, **production-ready architecture** designed for real-world business requirements.



<br>



**Technical Highlights:**

* **Clean Architecture:** Strict separation of concerns (Domain/Data/Presentation).

* **Offline-First:** Resilient data handling using **Room** as SSOT.

* **Solid Foundations:** Native **Java** implementation with **MVVM** & **LiveData**.

* **Reliability:** 100% Business Logic coverage with **JUnit & Mockito**.
